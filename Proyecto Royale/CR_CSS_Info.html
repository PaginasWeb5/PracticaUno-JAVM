¿Que es CSS?

Cascading Stylesheets es un lenguaje de diseño gráfico para definir y crear la presentación
de un documento estructurado escrito en un lenguaje de marcado.​ Es muy usado para establecer
el diseño visual de los documentos web, e interfaces de usuario escritas en HTML o XHTML;
el lenguaje puede ser aplicado a cualquier documento XML, incluyendo XHTML, SVG, XUL, RSS, etcétera.
También permite aplicar estilos no visuales, como las hojas de estilo auditivas.

Mas que nada, sirve para decorar tus páginas web, desde efectos simples, ¡hasta las animaciones más complicadas!



No es necesario tener que crear todo un archivo para guardar todos tus "estilos" ya que con
el formato HTML basta con algunas cosas. Por ejemplo:

<p style="color:brown">

Hará que el color del texto sea café.

A veces, quieres cambiar muchas cosas, y no cabrá todo en una sola hoja, asi que llega CSS al rescate!
Nadamas creas un archivo de CSS, pones todo el codigo ahi, lo vinculas al HTML que tu quieras, y listo!

Para hacer eso, solo pones:
<link rel="stylesheet" href="Ejemplo.css"> dentro de la pagina HTML, adentro del head y ya.



Estilos

Fondos: Primero declaras un elemento específico, si así lo deseas, quieres modificar. Por ejemplo:

.body{

background-color: light-blue;

}

Y el fondo será azul ligero!

Bordes: Como anteriormente se hizo, se declara el elemento:

.p{

border-color: red; border-style: solid;

}

Y así se cambian los bordes!



Texto Animado: Para hacer esto se requiere tener un poco de conocimiento sobre las animaciones en general.

Depende de que color quieres, si lo quieres de izquierda a derecha, gradiente o no, todo a la vez o de letra por letra.

Aun así, nada es imposible!
Si quieres tener el efecto que puse de ejemplo, primero hay que declarar un @keyframes.
Esta es la base para realizar cualquier animación que deseas.

Como vamos a hacer el efecto un gradiente que afecte letra por letra, tendremos que declarar
cada color en cada "fase" de la animación. Haremos que dure 6 segundos y que se repita por un número indefinido de veces.

@keyframes colores{

0% {color:#D68AB0;}
25% {color:#D8CE5D;}
50% {color:#6BD85D;}
75% {color:#5DB9D8;}
100% {color:#D85DCB;}

}

Despues de declarar, entonces pondremos el nombre del elemento (en este caso colores), para que
cada vez que se pide a llamar el elemento "legendaria", aparecerá el texto animado!

.legendaria{

background-size: 1000%, 1000%;
background-clip: text;
animation-duration: 6s;
animation-timing-function: ease;
animation-delay: 0s;
animation-direction: normal;
animation-fill-mode: none;
animation-iteration-count: infinite;
animation-play-state: running;
animation-name: colores;
font-weight:bold

}

Si todo lo hiciste bien, tu texto se verá así!
