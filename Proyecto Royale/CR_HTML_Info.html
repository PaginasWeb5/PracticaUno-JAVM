¿Que es HTML5?

HyperText Markup Language version 5, hace referencia al lenguaje de marcado
para la elaboración de páginas web. Es un estándar que sirve de referencia del software
que conecta con la elaboración de páginas web en sus diferentes versiones,
define una estructura básica y un código (denominado código HTML) para la definición
de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

EL HTML5 es la version mas reciente de todas, salió en el 2014.

Sirve mas que nada para poder hacer sitios web y modificarlos a tu gusto!
Se pueden hacer una infinidad de cosas con la programación, y HTML
no es una excepción.



Todo elemento de HTML se describe con una etiqueta:

<head> Este es tu encabezado. </head>

De esa manera se hace toda la programación, aunque no todas las etiquetas tienen pares,
por ejemplo esta:

<br>

Esta es una etiqueta que te permite poner un espacio entre líneas.

A continuación se muestran las etiquetas básicas:

<html>: Es el elemento básico de una página html.

<head>: Es un elemento que contiene información principal de un documento, este es invisible al navegador.

<title>: Es un elemento que especifíca el titulo del documento. Se encontrará en la pestaña del navegador.

<h1>: Este es la etiqueta que se usa para poner títulos.

<body>: Aqui es donde esta presente TODO el contenido de un sitio web.





Atributos

Como cualquier cosa, todo se puede cambiar, existen muchas maneras de diseñar
tu pagina a tu gusto! Veremos como en esta sección:

<img src="Flores.jpg" alt="Flores">: Como su nombre lo indica, se usa para poner imágenes en tu pagina. En este caso, no se pone nada adentro de la etiqueta, sino dentro de las flechas.

En la parte de src se pone la direccion completa de la imagen que quieres poner, adentro de las
comillas, y en la parte de alt se le pone un nombre de referencia. Esto sirve en caso de que no se
pueda cargar la imágen.

<a href="https://www.google.com"> ¡Haz clic aqui! </a>: Esta etiqueta sirve para proporcionar links a otros sitios web.
En la parte de href, pones el link donde quieres que el usuario vaya a visitar.
En la parte de adentro, pones el texto o una descripción del link, lo que sea necesario.

Hasta puedes poner el texto de un link en la parte del contenido para que parezca que irás
a una página, ¡y poner otro link dentro del href!





Tipos de Texto

Como muchos editores de texto, también se puede modificar el texto de
HTML. Estas son algunas de las etiquetas usadas:

<b>: Texto en negritas.

<i>: Texto en itálico.

<em>: Texto con énfasis.

<mark>: Texto enmarcado.

<small>: Texto pequeño.

<del>: Texto eliminado.

<ins>: Texto insertado.

<sub>: Texto inferior.

<sup>: Texto superior.
